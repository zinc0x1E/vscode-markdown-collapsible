# Markdown-Collapsible

Add `<details>` and `<summary>` support to VS Code's built-in Markdown preview via *markdown-it-collapsible*

## Feature

```markdown
+++ **Click here to expand**
  Here is some *hidden* content
  +++ Nesting is supported as well
    Hidden Text
+++
```
